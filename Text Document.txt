1. Export.csv - файл с исходными данными, загружаемыми в БД
2. Export.sql - скрипт для создания БД
3. app.py - код приложения на Python

1/ Необходимо установить файлы в одну папку на компьютер пользователя.
2/ Создать базу данных в pgAdmin, используя файл Export.sql
3/ В редакторе в строке 11 password="***" ввести пароль пользователя из pgAdmin 
4/ Запустить код
